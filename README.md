# Template

Use this template when starting a project with Node.
Included: Express, Mongoose, Docker, Jest

## Installation

```bash
npm install
```

## Running Dev Server

```bash
npm run start
```

## Running docker-compose

```bash
npm run docker
```

## Start mongo server from ./data folder

```bash
npm run mongo
```

## Build for development

```bash
npm run build:dev
```

## Build for production

```bash
npm run build:prod
```

## Setting Environment

```bash
npm run set:dev
npm run set:prod
```

## Start testing

```bash
npm run test
npm test
```

## Watch testing

```bash
npm run test:watch
```

## eslint

```bash
npm run eslint
```

## prettier

```bash
npm run prettier
```

## Clean dist folder

```bash
npm run clean
```
